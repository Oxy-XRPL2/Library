1. Menu Properties

This is the menu properties editor.
It helps you to configure menu properties like font, borders, colors, common icons, positioning and others.
To change property value click on the button .


2. Menu Items Tree View.




You can move menu items with a mouse.
To move the item up and down use "Move Up" and "Move Down" buttons on "Item Operations" panel.
To rename the item click them and hold the mouse cursor about a second.

3. Item Operations Controls.

Add Item - add a new item after selected item;
Add SubItem - add a sub item for selected item;
Insert Item - insert item after selected item;
Delete Item - delete selected item;
Delete All - delete all items;
Move Up, Move Down - move selected item up/down relatively the current position;
Copy Item - copy selected item to a clipboard;
Paste Item - paste selected item from a clipboard.


4. Selected Item Properties.

Text - item title;
Link - page (or resource) that appear after clicking the the item;
Target - link target;
Tip - item hint;
Images:
    Normal - normal item icon;
    Over - mouse over item icon;
Item Style;
Submenu Style - style of a submenu that contain selected item. Style works if the item is the first in submenu.


5. Individual Styles Settings

Item Styles - open the item styles window;
Submenu Styles - open the submenu styles window.


6. File Controls and Previewing Controls

New Script - reset all parameters;
Open Script - load .js file from a disk;
Save Script - open the save dialog;
Fullscreen Preview - open a single window for preview;
Preview in Browser - preview menu in a default browser.


7. Updating Process Controls

Auto Update - updating a preview after every change;
Update - update a preview immediately.


8. Preview Region

Region displays the menu.
Use the "Preview" checkbox to hide/show the peview region.


9. Hint Region

Regions displays a short info about selected menu property.
